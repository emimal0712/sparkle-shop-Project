# Sparkle's Shop - E-commerce Web Application

## Project Description
Sparkle's Shop is a basic full-stack e-commerce web application developed as part of my Java Full Stack Development training. The project demonstrates frontend and backend integration using Spring Boot, React.js, and MySQL.

## Features
- User-friendly interface
- Product listing
- CRUD operations
- REST API integration
- Responsive design
- MySQL database connectivity

## Technologies Used
- Java
- Spring Boot
- React.js
- JavaScript
- HTML
- CSS
- Bootstrap
- MySQL
- Git & GitHub
- Postman

## How to Run

### Backend
1. Open the backend project in IntelliJ IDEA.
2. Configure MySQL database.
3. Run the Spring Boot application.

### Frontend
1. Open the frontend project.
2. Run:
```bash
npm install
npm run dev
```
